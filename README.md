# 🧳 MayTrip

첫 여행이라, 바쁘게 사느라 여행 계획이 막막한 사용자들의 여행 동행자 찾기의 고민, 나의 여행 기록까지 한번에 해결해주는 여행 앱

## 주요 기능

- 여행 루트 생성 및 공유
- 커뮤니티 게시판
    - 질문
    - 여행 후기 작성
    - 동행인 구하기
- 채팅
